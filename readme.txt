A widget that easily let your site's visitors connect with your presence in social media.

== Description ==

Connect With Me offers a widget to place in any sidebar of your theme. Within the widget is a list with icons that link to your/your site's profile of the social media sites and communities (e.g. Facebook, Twitter, LinkedIn) of your choosing.

You can easily manage which social media sites and communities you wish to include in the list, and you can select from various sets of icons to make the list suit your theme.

== Features ==
* Currently supports 8 easy-to-manage connections or profiles: Facebook, Twitter, LinkedIn, Google Plus, Flickr, YouTube, Skype, E-mail
* 3 different sets of icons to choose from for adaptibility to your theme.
* Sidebar Widget to easily place the list of links and icons on you blog or website

== Installation ==

1. Download the plugin from <a href="http://wordpress.org/extend/plugins/connect-with-me/">WordPress.org Plugin Repository</a>
2. Unzip the file
3. Upload the folder *connect-with-me* to the */wp-content/plugins/* directory of your WordPress installation
4. Activate the plugin through the "Plugins" menu in WordPress
5. Go to the plugin's settings page under Settings -> Connect With Me
6. Enter the information for every social media you want to link to. Empty fields indicate that the link/icon should be excluded from the list. Click "Save" when finished.
7. Go to your widget administration under Appearance -> Widgets
8. Drop the "Connect With Me" widget in the sidebar of your liking, enter a title, and save it - or, place the code `<?php wperConnectWithMe(); ?>` in your theme template!
